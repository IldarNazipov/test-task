# Онлайн-курсы

Страница с онлайн-курсами (каталог с поиском и фильтрами).

## Описание

- **Резиновая верстка** без брейкпоинтов, плавное масштабирование шрифтов, отступов и элементов
- **Поиск по названию карточек** (живой, с дебаунсом 300мс)
- **Фильтрация по категориям**
- Верстка по методике [БЭМ](https://ru.bem.info/methodology/)

## Стек технологий

- HTML
- SCSS
- Vanilla JavaScript

Без использования библиотек и фреймворков.

Демонстрация: https://test-task-green-zeta.vercel.app/
